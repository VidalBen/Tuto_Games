    Sobre este chat

    Regras rápidas

    • Nome de usuário até 10 caracteres — termos proibidos viram ########.
    • O mesmo filtro vale para o texto da mensagem: trechos bloqueados são substituídos por ####.
    • Não compartilhe dados pessoais; use Denunciar se precisar.

    Figurinhas

    Toque nas figurinhas abaixo do campo de mensagem para inserir emojis no texto — combine com palavras ou mande só uma carinha para reagir.

    Onde isso fica salvo?

    Por padrão o histórico fica neste navegador. Com Supabase configurado em fale-conosco-config.js, as mensagens podem sincronizar entre dispositivos.

    Sala de bate-papo

    Escolha um nome de usuário com até 10 caracteres. Nomes com palavras impróprias ou conteúdo adulto são trocados automaticamente por ########.

    Figurinhas — toque para inserir no texto

    Teste: apaga o histórico local. O botão só fica ativo depois da senha de administrador em Senha adm (rodapé).

    Denúncias ficam registradas neste navegador até termos moderação em servidor. Tabela sugerida no Supabase: public_chat_messages (username, body, created_at).
